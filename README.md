## 🚀 What this app does

live-workshop-skill helps you turn a lecture, workshop, or meeting into a clear knowledge base. It uses live Whisper transcription to capture speech as text, then helps organize that text so you can find key points later.

Use it when you want to:
- Follow a talk without taking notes by hand
- Search past lectures for a topic or phrase
- Keep a record of class or workshop content
- Review important details after a session ends

## 💻 What you need

Before you install the app, make sure you have:
- A Windows PC
- An internet connection for the download
- A microphone if you want live transcription
- Enough free space for the app and saved transcripts
- Windows 10 or Windows 11

For best results:
- Use a headset or external mic if you can
- Close apps that use your microphone
- Keep your speakers at a steady level during transcription

## ▶️ How to use it

1. Open the app
2. Start your lecture, class, or meeting
3. Click Start Transcription
4. Speak normally in the room
5. Let the app capture the live audio
6. Stop transcription when the session ends
7. Save the transcript
8. Review or search the text later

A good workflow:
- Start the app before the lecture begins
- Keep the mic close to the speaker
- Save each session with a clear name
- Use dates in file names, like `history-class-2026-04-14`

## 🎛️ Tips for better transcription

Whisper works best when the audio is clear. Try these tips:

- Place the mic near the speaker
- Keep background noise low
- Avoid two people talking at once
- Speak at a steady pace
- Pause between topics
- Test the mic before the session starts

If text looks wrong:
- Check the mic selection
- Lower room noise
- Move closer to the speaker
- Try a different language setting if needed

## ❓ Troubleshooting

### The app does not open
- Check that the download finished
- Try opening the file again
- Right-click and choose Run as administrator
- Re-download the file from the release page if the file looks broken

### No audio is being captured
- Check your microphone connection
- Make sure the mic is not muted
- Select the correct input device in the app
- Check Windows sound settings

### The transcript is empty
- Speak into the mic
- Move the mic closer
- Turn up input volume
- Make sure transcription is started

### The text looks wrong
- Reduce noise in the room
- Use a better mic
- Speak in shorter phrases
- Check the language setting

### Windows blocks the file
- Open the file properties
- If you see an Unblock option, select it
- Try running the app again
- Download only from the release page link above
